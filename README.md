# Flutter Products Managing Web App

This is a Flutter-based web application that integrates with Firebase for authentication, hosting, and Firestore database to manage products. The app allows users to register, log in, view products, and manage products.

## Features
- **User Authentication:** Firebase Authentication is used for user registration and login.
- **Product Management:** Users can view and manage products.
- **Firestore Database:** Products and user information are stored in Firebase Firestore.
- **Firebase Hosting:** The app is hosted on Firebase Hosting for easy deployment.

You can visit the web app using this **[link](https://my-firebase-project-b1913.web.app/)**
